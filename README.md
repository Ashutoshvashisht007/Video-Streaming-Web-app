# Video-Streaming-Web-app
Live Website link: https://youtubebackend.onrender.com/

In this web application, a user can add videos, like them, dislike them and can even subscribe to other channels.
The user can also see the trending videos and can sigin using google id.

The tech stack used in this project is MERN stack where M stands form MongoDB, E stands for Express, R stands for React.js and N stands for Node.js.
